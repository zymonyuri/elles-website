# Elle's Flower Boutique

Elle's Flower Boutique is a romantic floral website designed to showcase bouquet arrangements in a soft, elegant, and feminine style. It features a curated collection of flower bouquets, an about page sharing the story behind the brand, and an order flow that lets visitors browse arrangements and view checkout details.

The website is built as a lightweight static site using HTML, CSS, and JavaScript. Its design centers on a refined floral aesthetic, using warm champagne, blush, and deep cocoa tones inspired by the brand logo. Across the site, the layout focuses on graceful typography, delicate spacing, romantic imagery, and a cohesive boutique feel suitable for web deployment.

## Features

- Homepage with featured bouquets
- Full arrangements collection page
- About page with brand story and history
- Product order and checkout pages
- Romantic, elegant, and brand-cohesive UI
- Responsive layout for desktop and mobile
- GitHub Pages-friendly file paths and asset usage
